   - 깔끔하고 전문적인 디자인

## 🎯 승인 후 할 일

### 1. 새 API 키 확인

```bash
# 마이페이지 → 활용신청 현황에서 확인
운영계정 API 키: xxxxx-운영계정용-xxxxx
```

### 2. .env 파일 업데이트

```bash
# .env 파일 열기
nano .env

# BOKJIRO_API_KEY를 새 운영계정 키로 변경
BOKJIRO_API_KEY=새로운_운영계정_API_키
```

### 3. 전체 파이프라인 실행

```bash
# 가상환경 활성화
source venv/bin/activate

# 전체 파이프라인 실행 (데이터 수집 + 임베딩)
python scripts/run_full_pipeline.py
```

**예상 소요 시간:**
- 중앙부처 수집: ~30분 (2,000건)
- 지자체 수집: ~1시간 (5,000건 × 17개 시도)
- 임베딩 생성: ~2시간 (병렬 처리)
- **총 소요: 약 3-4시간** ⏱️

### 4. 데이터 검증

