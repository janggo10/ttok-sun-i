## 📝 서비스 설명 (4000 bytes)

```
[서비스 개요]
'또그쎤이'는 카카오톡 챗봇 기반의 AI 맞춤형 복지혜택 추천 서비스입니다. 
복잡한 복지 제도를 일반 국민이 쉽게 찾고 신청할 수 있도록 돕는 것이 목표입니다.

[배경 및 필요성]
대한민국에는 7,000개 이상의 복지 혜택이 있지만, 정보 접근성 부족으로 인해 
많은 국민이 자신에게 맞는 혜택을 찾지 못하고 있습니다. 특히 고령층이나 
저소득층은 복잡한 정부 웹사이트 이용에 어려움을 겪고 있으며, 
이로 인한 '복지 사각지대' 문제가 심각합니다.

[주요 기능]
1. 개인 맞춤형 복지 추천
   - 사용자의 거주지역, 연령대, 가구상황을 바탕으로 적합한 복지혜택 추천
   - 중앙부처 및 지자체 복지 정보 통합 제공
   - 실시간 데이터 업데이트로 최신 정보 유지

2. AI 기반 시맨틱 검색 시스템
   - 자연어 질의응답: "출산 지원금 있나요?" 같은 일상 대화로 검색
   - Hybrid RAG (Retrieval-Augmented Generation) 기술 적용
   - 구조화된 필터링(지역, 연령대, 생애주기)과 벡터 검색 결합
   - AWS Bedrock Titan Embeddings를 활용한 고도화된 자연어 이해

3. 카카오톡 챗봇 연동
   - 별도 앱 설치 불필요, 카카오톡으로 접근 가능
   - 대화형 UI로 복지 정보 탐색 및 신청 안내
   - 복지로(bokjiro.go.kr) 등 신청 사이트 연계
   - 24시간 자동 응답 서비스

4. 포괄적 복지 정보 제공
   - 지원대상, 지원내용, 신청방법, 신청기간 상세 안내
   - 문의처 정보 제공으로 추가 상담 연계
   - 첨부서류 및 신청 절차 안내

[활용 공공데이터]
- API 명: 복지서비스 목록조회 (NationalWelfarelistV001, LcgvWelfarelist)
- API 명: 복지서비스 상세조회 (NationalWelfaredetailedV001, LcgvWelfaredetailed)
- 제공기관: 보건복지부 (한국사회보장정보원)
- 활용 데이터: 중앙부처 복지정책, 지자체 복지정책 전체

[데이터 수집 및 관리 계획]
1. 초기 전체 수집
   - 중앙부처 복지혜택: 약 2,000건
   - 지자체 복지혜택: 약 5,000건 (17개 시도)
   - 상세정보 포함 완전한 데이터베이스 구축

2. 정기 업데이트
   - 주 1회 전체 데이터 갱신 (변경사항 반영)
   - 일 1회 신규 및 변경 데이터 증분 수집
   - 종료된 혜택 자동 비활성화 처리

3. 데이터 품질 관리
   - 중복 제거 및 데이터 정합성 검증
   - 벡터 임베딩 생성으로 검색 정확도 향상
   - 로그 기록 및 오류 모니터링

[기술 스택]
- Backend: Python, AWS Lambda (서버리스)
- Database: Supabase (PostgreSQL + pgvector)
- AI/ML: AWS Bedrock, Titan Embeddings V2
- Chatbot: Kakao i Open Builder
- Infrastructure: AWS (Lambda, Step Functions, EventBridge)

[기대 효과]
1. 사회적 가치
   - 복지 사각지대 해소: 몰라서 못 받는 복지혜택 방지
   - 정보 접근성 향상: 디지털 소외계층도 쉽게 이용
   - 복지 체감도 상승: 신속한 정보 제공으로 만족도 개선

2. 행정 효율성
   - 복지 담당자의 반복 문의 응대 부담 경감
   - 자동화된 안내로 행정 비용 절감
   - 데이터 기반 복지정책 수립 지원

3. 경제적 효과
   - 복지 예산의 효율적 집행
   - 대상자 발굴률 향상
   - 연간 예상 이용자: 10만명 이상

[개발 현황 및 일정]
- 2025년 12월: 프로젝트 기획 및 설계 완료
- 2026년 1월: 데이터 수집 파이프라인 구축 완료
- 2026년 1월: Vector DB 및 RAG 시스템 구축 완료
- 2026년 2월: 카카오톡 챗봇 연동 및 베타 테스트
- 2026년 3월: 정식 서비스 출시 예정

[운영 계획]
- 초기 사용자 피드백 수집 및 개선
- 지속적인 AI 모델 정확도 향상
- 추가 공공데이터 연계 (일자리, 주거, 교육 등)
- 지자체와 협력하여 맞춤형 복지 안내 서비스 확대

본 서비스는 공공데이터를 활용하여 국민의 복지 접근성을 높이고, 
디지털 기술로 사회적 가치를 창출하는 것을 목표로 합니다.
```
