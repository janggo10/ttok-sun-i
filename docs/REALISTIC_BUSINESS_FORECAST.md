## 7. 유튜브 숏폼 자동화 가이드

### 7.1 추천 기술 스택

```
콘텐츠 생성 파이프라인:

┌─────────────────────────────────────┐
│ 1. DB에서 신규 혜택/일자리 선별      │
│    - 하루 1개 선택 (긴급도/중요도)   │
│    - SQL: SELECT * FROM benefits     │
│      WHERE created_at > NOW() - 1    │
└─────────────────────────────────────┘
            ↓
┌─────────────────────────────────────┐
│ 2. Claude API로 대본 생성           │
│    - 프롬프트 엔지니어링:           │
│      "30초 유튜브 숏폼 대본 작성"   │
│    - Hook(3초) → Problem(10초)      │
│      → Solution(10초) → CTA(7초)   │
│    - 비용: $0.003/요청              │
└─────────────────────────────────────┘
            ↓
┌─────────────────────────────────────┐
│ 3. TTS 음성 생성                    │
│    Option 1: ElevenLabs             │
│      - 자연스러운 한국어            │
│      - 비용: $22/월 (30만 글자)     │
│    Option 2: OpenAI TTS             │
│      - 비용: $0.015/1K 글자         │
│      - 한국어 품질 괜찮음            │
└─────────────────────────────────────┘
            ↓
┌─────────────────────────────────────┐
│ 4. 이미지/영상 소스 수집            │
│    Option 1: Pexels (무료)          │
│      - 저작권 걱정 없음             │
│    Option 2: Canva Pro ($13/월)     │
│      - 템플릿 풍부                  │
│    Option 3: Midjourney (AI 생성)   │
│      - 독창적이지만 비용 높음       │
└─────────────────────────────────────┘
            ↓
┌─────────────────────────────────────┐
│ 5. 자동 편집 (Python)               │
│    - FFmpeg (커맨드라인)            │
│    - moviepy (Python 라이브러리)    │
│    - 자막 자동 삽입                 │
│    - BGM 추가 (로열티프리)          │
└─────────────────────────────────────┘
            ↓
┌─────────────────────────────────────┐
│ 6. 자동 업로드                      │
│    - YouTube Data API v3            │
│    - Make.com으로 스케줄링          │
│    - 매일 오전 9시 자동 업로드      │
└─────────────────────────────────────┘
```

**총 월 비용: 약 15-20만원**

- 즉시성 강조

### 8.2 영업 실전 팁 (경험 없어도 OK)

- 거절은 배제가 아니라 필터링

#### 2. 숫자로 말하기
- "효과 좋습니다" ❌

- 무료 제휴로 3-5건 성공 사례 만들기
- 후기/추천서 받아두기

#### 4. 작은 것부터 시작
- 대형 업체보다 소규모 업체가 빠름
- 1-2곳 성공하면 레퍼런스로 활용

#### 5. 지속적 팔로업
- 1차 거절해도 3개월 후 재연락
- 가입자 증가하면 설득력 높아짐
